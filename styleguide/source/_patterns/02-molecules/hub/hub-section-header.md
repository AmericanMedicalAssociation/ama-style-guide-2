### Description
The Section Hub Header separates the the hub page into relevant sections.

~~~
"hubSectionHeader": {
  heading {
    level:
      type: string / required
    text:
      type: string / required
    class:
      type: string / optional
  }
  paragraph {
    text:
      type: string
    class:
      type: string / optional
  }
}
~~~
