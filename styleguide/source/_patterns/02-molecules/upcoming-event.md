### Description
A teaser of the event that will appear in a listing on the homepage.

### Variables
~~~
event: {
  link: {
    title: string,
    href: string,
    text: string,
    target: _self,
    class: ama__link--icon,
    iconLeft: @atoms/media/icons/svg/icon-calendar.twig
  },
  location: {
    text: string,
    class: ama__upcoming-event__location
  },
  date: string/optional
}
~~~
