### Variant options

### Variables
~~~
button: {
  href:
    type: string (url) / optional
  info: 
    type: string / optional
  text: 
    type: string / required
  type:
    type: string / optional (ex: "button", "submit")
  style:
    type: string / optional ("secondary" or "")
}
~~~
